Task 4: Dashboard Design

Objective

To design an interactive Power BI dashboard that provides insights into sales performance, profit trends, and category-wise analysis for business stakeholders.

Dataset Used

Superstore Sales Dataset (Kaggle)

Steps Performed

• Imported the dataset into Power BI

• Verified and cleaned data fields

• Created KPI cards (Total Sales, Total Profit, Sales Growth %, Total Orders)

• Built line,bar,table, column, donut, and tree map visuals for analysis

• Added slicers for Region, Category, and Order Date

• Organized visuals into two pages: Sales Overview and Sales Analysis

• Applied clean formatting and navigation for better usability

Tools Used

• Power BI Desktop

Output

• Interactive Sales & Profit Analysis Dashboard
